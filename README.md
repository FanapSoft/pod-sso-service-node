# Pod Billing Service

This Module Contains All the methods related to POD Singlr Sign On (SSO)
These Methods Are:

## Web Related Methods:
* getLoginUrl
* getAccessToken
* refreshAccessToken
* getTokenInfo
* revokeToken

## OTP Related Methods:
* handshake
* authorize
* getOtpScenario
* verify
* getAccessTokenByOtp
* getAccessTokenByOtpScenario
